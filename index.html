drive.player

Reprodutor de música para o teu Google Drive. Toca pastas inteiras, cria playlists.

Configuração inicial (uma vez só):

1. Vai a console.cloud.google.com
2. Cria projeto → APIs → ativa Google Drive API
3. Credenciais → OAuth 2.0 → tipo Web application
4. Em "Authorized JavaScript origins" adiciona a URL deste ficheiro (ou http://localhost se correres local)
5. Copia o Client ID e cola abaixo
